快速入门

linux安装docker

https://docs.docker.com/engine/install/ubuntu/

基本操作命令

查看镜像

docker images

查看启动的容器

docker ps -a

关闭容器并删除

docker stop [容器名/容器ID]
docker rm [容器名/容器ID]

启动容器

docker run -name [容器名] -p xxxx:xxxx -d [镜像名称]

docker run –name mysql -p 3306:3306 -d mysql –MYSQL_ROOT_PASSWORD=”密码”

启动容器命令

mysql

docker run –name mysql -e MYSQL_ROOT_PASSWORD=xxx -d mysql

redis

docker run –name redis -p 6379:6379 -d redis –requirepass “密码”

postgresql

docker run –name postgre -e POSTGRES_PASSWORD=xxx -p 5432:5432 -d postgres

rabbitmq安装的相关问题

  • 启动容器，没有显示界面(开启插件)

    docker exec -it [容器名] bash

    rabbitmq-plugins enable rabbitmq_management

设置用户权限

进入命令行：docker exec -it some-rabbit bash

设置权限：rabbitmqctl set_permissions -p / Lyblue “.“ “.“ “.*”
