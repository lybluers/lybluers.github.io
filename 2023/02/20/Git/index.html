Git代码托管服务

安装

1.下载安装程序,安装,右键出现git选项即安装成功

2.基本配置

a.设置用户名和密码

b.使用 git ~/.bashrc 在用户目录创建一个.bashrc的文件,里面配置简化指令

随后在git bash使用

c.在 git目录/etc下的 bash.bashrc 文件中最后加入

3.配置本地仓库,新建文件夹将其设为git本地仓库,在其内部打开git bash,输入git init即可

配置gitee/github

1.使用公私钥对(直接在git bash使用)

2.查看密钥并复制

3.进入码云进行相应设置,粘贴公钥和标题,输入密码,配置完成

4.测试连接

连接远程仓库

1.在码云创建仓库

2.复制ssh地址

3.连接本地仓库和远程仓库(连接名称默认为origin)

  • 远程连接名称:默认为origin,取决于远程服务器设置,可以自行设置
  • 仓库路径:从远程仓库获取的地址(如SSH地址)

查看本地仓库的远程连接名称

4.将本地的代码推送到远程仓库上

tip:-f 表示强制覆盖
