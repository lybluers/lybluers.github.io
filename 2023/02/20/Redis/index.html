简介

基本操作

Ubantu安装Redis

redis配置文件位置

/etc/redis

设置密码(/etc/redis/redis.conf)

requirepass 密码

关闭/开启服务

/etc/init.d/redis-server stop
/etc/init.d/redis-server start

启动redis

redis-cli -h localhost -p 6379 -a 密码

登录验证

127.0.0.1:6379> auth 密码

切换库

redis默认会有16个库

select 0 （0为索引值）

远程连接

  1. 将protected-mode yes 修改为no 关闭protected-mode模式，此时外部网络可以直接访问

    注释bind 127.0.0.1 只允许本机连接设置

    启动redis服务/etc/init.d/redis-server start

  2. 本地启动redis服务（在安装目录下打开redis-server.exe）

在当前目录下打开powershell，连接远程服务器redis

./redis-cli -h 120.25.248.122 -p 6379 -a 密码
