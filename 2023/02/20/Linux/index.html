基本命令

目录操作

进入文件地址

cd /xxx/xxx

创建文件夹

mkdir 文件名

新建文件

touch 文件名

查看文件内容

cat 文件名
more 文件名（分页展示）
tail [-f] 文件名

移动文件（更改文件名）

移动：mv 文件名 路径

重命名：mv 旧文件名 新文件名

删除文件

rm 文件名
rm -f 删除文件
rm -rf 删除文件夹及其下的所有文件

tip: -r 就是向下递归,不管有多少目录一并删除

-f 就是直接强行删除,不做任何提示

解压文件

jar 文件

jar -xvf xxx.jar(将内容解压，不创建文件夹)

用户管理

创建新用户(未授予root权限,无法查看root文件夹,移动或删除文件)

adduser 用户名

修改(创建)密码

passwd 用户名

查看所有用户信息:

信息文件在 /etc/passwd 中

加密密码文件在 /etc/shadow 中( 加密算法一般为 sha512 )

查看程序是否在运行

ps -ef | grep nginx

vi操作

退出模式

以指定文件名保存

:w filename

保存并退出

:wq

不保存直接退出

:q!

删除

删除光标所在整行

dd

复制粘贴

复制当前行

yy

粘贴

p

查找

列出行号

:set nu

其他

取消高亮

:nohl

MySQL操作

基本操作

查看数据库状态

sudo service mysqld status

启动服务

sudo service mysqld start

停止服务

sudo service mysqld stop

重启服务

sudo service mysqld restart
