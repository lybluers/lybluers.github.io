NPM

npm 是js的包管理器，类似于java的maven

部署环境

  1. 检查镜像

npm get registry

如果不是淘宝镜像，需要进行镜像设置

npm config set registry https://registry.npm.taobao.org/

  1. 搭建简单服务（进入需要创建环境的文件夹）

    npm install express –save-dev

创建出三个文件

  • node_modules:包库
  • package.json:相当于java当中的pom文件，用来添加依赖

  1. 修改package.json文件，使得express函数可以被调用
